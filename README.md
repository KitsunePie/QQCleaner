# 瘦身模块(QQCleaner)

2.0版本重构中~

[Telegram频道](https://t.me/QQCleaner)

注:本模块完全免费且开源，一切开发旨在学习，请勿用于非法用途。

## 使用方法

1.勾上模块     
2.重启QQ/TIM/微信   
3.设置->关于->QQ/TIM/微信瘦身

## 功能

1.自定义瘦身(导入/导出瘦身配置文件)     
2.定时瘦身(自动执行瘦身操作)
